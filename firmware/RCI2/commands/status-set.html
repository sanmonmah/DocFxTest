Set status command

This command specifies a status value of a determined system component. The componentType, Index1 and Index2 define which status value will be set.
CommandID 2 Command length: 10
Description: Specifies a status value of a determined system component.
Parameter Type Range Default
Input: ComponentType uint16 0x0-max_component_type -
Index1 uint16 0x0-0xFF -
Index2 uint16 0x0-0xFF -
Status1 uint32 0x0-0xFFFFFFFF -
Status2 uint32 0x0-0xFFFFFFFF -
Output: - - - -
